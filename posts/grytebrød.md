|            |                            |
| ---------- | -------------------------- |
| 450 g      | hvetemel                   |
| 150 g      | sammalt hvetemel , finmalt |
| 1/3 ts     | tørrgjær                   |
| 1-1 1/2 ts | salt                       |
| 1 klype    | sukker                     |
| 5 dl       | vann , romtemperert        |

Miks alt i en bolle, dekk til bollen og la heve i 12-20 timer.
